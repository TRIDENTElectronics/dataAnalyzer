# dataAnalyzer
## 用法
将数据所在文件夹和所有脚本置于同一项目文件夹，打开所有脚本，运行main.py即可
## 功能目录
main.py脚本暂时简单地采用命令行交互以整合功能，目前整合的功能目录如下：
### 1线性拟合
I:需要键入两组直流电压值并依次选择其对应的.data数据文件读取\n
O:之后会自动保存线性拟合参数文件
### 2数据解包/分析绘图
I:需要选择目标文件，选择并加载线性校正参数文件（可选，同一文件之前加载过就不必再选）
O:之后会输出各类数据有效数据包数量/4chip各自DRS最小触发号（第一个数据包）
然后可以选择以下功能以继续：
#### 1全通道绘图
I:键入数据包触发号
O:之后会先后绘制符合此触发号的31通道（从0开始）原始码值波形/线性校正后波形
#### 2单通道绘图
I:键入数据包触发号，通道号（从0开始），（是否需要追加低通滤波（截止频率可改）），（是否需要保存csv文件）
O:之后会先后绘制单通道原始码值图/校正后波形/（滤波后波形），（并保存原始码值/校正后波形各为一个csv文件）
